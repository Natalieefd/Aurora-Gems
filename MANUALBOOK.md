<img src="https://github.com/Natalieefd/Aurora-Gems/blob/main/public/ss/landing_page.jpeg">

<h1 class="text-4xl font-bold ">
          Add Produk Admin 
</h1>
<img src="https://github.com/Natalieefd/Aurora-Gems/blob/main/public/ss/create.jpeg">
<img src="https://github.com/Natalieefd/Aurora-Gems/blob/main/public/ss/add_data_product3.png">
<h4 class="text-4xl font-normal "> 
    Pada saat admin menekan tombol "Create" di halaman manajemen produk, admin akan diarahkan ke halaman tambah produk (add produk). Proses ini memungkinkan admin untuk mengisi formulir dengan informasi yang diperlukan untuk menambahkan produk baru ke dalam sistem. Halaman "add produk" berisi formulir yang mencakup kolom-kolom seperti nama, kategori, deskripsi, size, berat, warna, stok, harga, dan gambar. Admin dapat mengisi setiap kolom sesuai dengan spesifikasi produk yang akan ditambahkan. Setelah mengisi formulir, admin kemudian dapat menyimpan data dengan menekan tombol "Submit". Desain formulir ini memberikan kemudahan dan kelengkapan bagi admin dalam proses penambahan produk baru.
